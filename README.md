# FriendFinder

##### Deployed On [Heroku](https://glacial-beyond-65720.herokuapp.com/)

Friend Finder is a full-stack web application that matches the user with the most compatible 'friend' based on their responses to the survey.

### Technologies Used:
```
- Express
- NodeJS
- HTML5
- Bootstrap
- Sweet Alert
```
